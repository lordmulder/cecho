Usage
-------------

`cecho.exe <forground_color> [<background_color>] <echo_text>`


Color names
-------------

`0 -->        black`
`1 -->        white`
`2 -->         grey`
`3 -->          red`
`4 -->     dark_red`
`5 -->        green`
`6 -->   dark_green`
`7 -->         blue`
`8 -->    dark_blue`
`9 -->       yellow`
`a -->  dark_yellow`
`b -->      magenta`
`c --> dark_magenta`
`d -->         cyan`
`e -->    dark_cyan`


Escape chars
-------------

`\n --> Line feed`
`\t --> Tabulator`
`\\ --> Backslash`

   
Example
-------------

`cecho.exe blue yellow "Hello world!\nGoodbye cruel world!"`
